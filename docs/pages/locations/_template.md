# Location Page Template

All location pages follow this structure. The goal is to rank for "[city/region] library
entertainer" searches while also giving local librarians confidence that Joe knows and
serves their area specifically.

---

## SEO Formula
**Title tag:** `Library Shows in [City/Region] | Oklahoma Library Shows — Funky Monkey Events`
**Meta description:** `[Joe/Funky Monkey Events] brings [#] interactive library programs to [City/Region]. [Local angle or travel fee note]. Book for your 2026 or 2027 summer reading program. (405) 431-6625.`
**H1:** `Library Programs in [City/Region]` (must include geographic keyword)
**Canonical:** `https://www.oklahomalibraryshows.com/locations/[slug]`

---

## Sections (in order)

### 1. Nav + CSLP Banner
Standard — same as all pages.

### 2. Location Hero
- **H1:** `Library Shows in [City/Region], [State]`
- **Hook:** One sentence specific to this location — travel fee status, local library systems, or unique angle
- **CTA:** "Check availability in [City/Region]" → booking modal
- **Travel fee badge:** either "No travel fee" (OKC/Tulsa metro) or "Travel fee may apply — contact for quote"

### 3. Serving [Location] Libraries
2–3 paragraphs that feel local:
- Mention specific library systems by name (Metropolitan Library System, Pioneer, EOLS, Tulsa City-County, etc.)
- Reference any known past shows at local libraries (if applicable)
- Note the travel situation honestly (metro = free, rural = bundling saves money)
- Oklahoma pages: mention ODL program listing

### 4. Shows Available in [Location]
Six show cards — same as homepage but abbreviated.
For out-of-state pages: note that all 6 shows are available with travel arrangements.
For rural OK: note that bundling multiple shows across a route reduces travel cost.

### 5. Why [Location] Libraries Choose Funky Monkey
3 trust bullets specific to location:
- "No travel fee within [metro]" (if applicable)
- "Flexible scheduling to accommodate your library system's summer week"
- "[X] libraries already served in [state/region]" (placeholder number — update with actuals)

### 6. Libraries We've Served in [Region]
List of library systems/branches served in this area.
If few known, keep section as placeholder: "Join the growing list of [region] libraries."
Link to service-area.html for full map.

### 7. Testimonial
One quote from a librarian in this region (or nearest match).
Use Kelsey Williamson (Norman/Pioneer — OKC area) or Jennifer Fuller (EOLS — Eastern OK) where appropriate.

### 8. All 6 Shows CTA Band
"Bring any of these programs to [Location]" + 6 show mini-cards
Each links to the full show page.

### 9. Book for [Location]
- CTA: "Check availability in [City/Region]" → booking modal
- Phone + email
- Response promise: "We respond within 24 hours"

### 10. Footer
Standard footer.

- Link to nearby location pages (e.g., Tulsa page links to Eastern OK and Rural OK)
