# Show Page Template

All six show pages follow this exact section order. Copy this template for each show.
See individual show spec files for page-specific content, keywords, and 2027 notes.

---

## SEO Formula

**Title tag:** `[Show Name] for Oklahoma Libraries | Oklahoma Library Shows`
**Meta description:** `[Hook with primary keyword]. [1 key detail — age, duration, or unique feature]. Book for your 2026 or 2027 summer reading program. Call (405) 431-6625.`
**H1:** Must contain the show name + "Oklahoma library" or "summer reading program"
**Canonical:** `https://www.oklahomalibraryshows.com/shows/[show-slug]`

---

## Sections (in order)

### 1. Nav
Standard nav — same as homepage.

### 2. CSLP Banner
Standard green banner: `2026 CSLP Theme: "Unearth a Story" | Now booking 2026 & 2027`

### 3. Show Hero
- Large H1 with show name
- One-line hook connecting to CSLP theme or librarian pain point
- Pill badges: duration | age range | indoor/outdoor | $385
- "Book this show" CTA → opens booking modal with this show pre-selected
- Hero photo (landscape, library setting if available)

### 4. Program Overview
2–3 sentences connecting the show to the CSLP theme.
Use the approved ODL description from references.md as the base.
Add one sentence about what makes this show distinctly library-appropriate.

### 5. What Happens
Narrative walkthrough of the show experience — present tense, second-person perspective
("Your audience will..."). 3–5 short paragraphs. Make it easy to picture.
This section should answer the question "what will my patrons actually experience?"

### 6. Educational Connections
- CSLP theme tie-in (literacy, STEAM, or creativity depending on the show)
- Learning outcomes (from ODL submission — listed as bullet points)
- One paragraph on why this fits a library's educational mission
- Brief note linking to 2027 "Mysteries Await" adaptation (link to 2027-summer-reading.html)

### 7. Show Details Grid
Display as a clean visual grid (2×3 or 3×2 layout):
| Detail | Value |
|--------|-------|
| Duration | 45–60 minutes |
| Age Range | [show-specific] |
| Audience Size | Up to [X] |
| Indoor/Outdoor | Indoor (or Outdoor Only for Foam Party) |
| Setup Time | 20–30 minutes |
| Price | $385 per program |

Add: "We accept purchase orders and checks. No credit card required."

### 8. Librarian Testimonials
2–3 quotes, each with:
- Quote text (specific to THIS show if possible)
- Name, title, library system
- Use real names from references.md; write realistic quotes if specific ones aren't available yet
- Photo placeholder (silhouette icon with library name if no headshot available)

### 9. Photo / Video Section
- 2–3 performance photos from this show (see photo-brief.md)
- Video embed placeholder: "Video coming soon — want to see it live? [Book a visit]"
- If a video exists: embed YouTube/Vimeo directly

### 10. "Also Consider" — Related Shows
2 other show cards (mini versions) to drive internal links.
Logic:
- Magic Show → also show Magic Workshop + Story Doodles
- Bubble Show → also show Magic Show + Foam Party
- Magic Workshop → also show Magic Show + Balloon Workshop
- Balloon Workshop → also show Magic Workshop + Story Doodles
- Story Doodles → also show Magic Workshop + Bubble Show
- Foam Party → also show Bubble Show + Magic Show

### 11. Book This Show — CTA Band
- Headline: `Ready to book [Show Name] for your library?`
- Supporting line: "We respond within 24 hours. No contracts, no credit cards."
- CTA: "Book [Show Name]" → opens booking modal with show pre-selected
- Secondary CTA: link to faq.html ("Have questions first?")

### 12. Footer
Standard footer — same as homepage.

- Update Section 6 (Educational Connections) with new theme tie-in
- Remove "2027 coming soon" language once show is confirmed
